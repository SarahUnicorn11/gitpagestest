How to use Git Pages

1. Create new repo on Github

2. git clone, then paste link

3. cd into new directory

4. Open text editor

5. Create/edit relevant files

6. Add changes, git add, commit, etc.

7. git push origin main

8. Refresh GitHub and check everything it there.

9. Settings > Pages > Change source to main and select root. Save.

10. Open link in new tab.
